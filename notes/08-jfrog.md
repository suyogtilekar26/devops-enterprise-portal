# JFrog Artifactory

## Definition

JFrog Artifactory is an enterprise artifact repository manager.

It stores:

- Docker Images
- Helm Charts
- Maven Packages
- npm Packages
- Python Packages

---

## Why do we use JFrog?

- Private repositories
- Security
- Version management
- Access control
- CI/CD integration
